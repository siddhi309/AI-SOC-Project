# 🛡️ AI-Powered Security Operations Center (AI-SOC)

## 📌 Project Overview
This project aims to build an AI-powered SOC platform that detects:
- Zero-day attacks
- Ransomware
- Phishing
- Insider threats

The system uses Machine Learning and behavioral analytics to analyze network logs and detect anomalies in real time.

---

## 👥 Team Structure

- 👩‍💻 Data Analysts – Data cleaning & preprocessing
- 🤖 ML Engineers – Model training & evaluation
- 🔐 Cybersecurity Engineer – Threat detection logic & attack analysis

## 📊 Dataset Information

We are using the CICIDS2017 dataset for intrusion detection research.

⚠️ The dataset is NOT uploaded to GitHub because it is large.

Each team member must download it manually and place it in:

data/raw/

---

## 🚀 How to Setup

1. Clone the repository:
   git clone https://github.com/siddhi309/AI-SOC-Project.git

2. Install dependencies:
   pip install -r requirements.txt

3. Download dataset and place inside:
   data/raw/

4. Run preprocessing notebook or script.
